Fonction

Définition

  • Une fonction associe une séquence d'instructions à un nom

  • La définition commence par un mot-clé def, puis d'une parenthèse et un deux points

  • Les fonctions peuvent contenir aucun, un ou plusieurs paramètres désignés par une variable. Les paramètres sont entre parenthèses et séparés par une virgule

  • Le mot-clé return renvoit le résultat d'une fonction

  • Une fonction peut renvoyer une valeur de base (entier, booléens...) dans le cas où une seule valeur est renvoyée

  • Une fonction renvoye un tuple dans le cas où plusieurs valeurs sont renvoyées
